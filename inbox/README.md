# Inbox — файлы от Павла

Сюда закидывай любые файлы: фото, документы, прайсы, таблицы, референсы.
Я буду их читать и использовать в работе.

## Как загрузить

1. Открой папку `inbox` на GitHub
2. Нажми **Add file → Upload files**
3. Загрузи файл, напиши commit message
4. Напиши мне в чате что загрузил и зачем — я прочитаю

## Подпапки

- `inbox/hairlove/` — всё по бренду HairLove (прайсы, материалы, фото)
- `inbox/products/` — фото продуктов, описания, составы
- `inbox/competitors/` — материалы по конкурентам
- `inbox/refs/` — референсы, вдохновение, примеры

## Форматы

PDF, DOCX, TXT, MD, JPG, PNG, XLSX, CSV и другие.
