```
/hairlove-site [задача]
```

Варианты:
```
/hairlove-site старт              ← с чего начать, если сайта нет
/hairlove-site структура          ← какие страницы нужны и зачем
/hairlove-site текст главная      ← написать текст для главной страницы
/hairlove-site карточка Lamellar Water   ← текст карточки продукта
/hairlove-site о нас              ← написать страницу "О бренде"
/hairlove-site SEO                ← ключевые запросы и метатеги
/hairlove-site платформа          ← какую платформу выбрать
```

**Без аргументов** — показываю план создания сайта с нуля: что делать первым, вторым, третьим.
