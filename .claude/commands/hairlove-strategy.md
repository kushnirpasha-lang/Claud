## Что выдаю

### Контент-план (на неделю/месяц)
- Конкретные темы постов с датами
- Форматы: Reels / карусель / фото
- Рубрики: польза / продукт / lifestyle / отзыв
- Оптимальное время публикаций

### Стратегия роста
- Конкретные действия, не теория
- Приоритеты по результату / усилию
- Метрики: что отслеживать

### Запуск продукта
- Прогрев аудитории за 1-2 недели
- День запуска: контент + механика
- Пост-запуск: поддержка продаж

### Воронка через контент
```
Охват (Reels, хэштеги) 
→ Вовлечение (полезный контент, сторис) 
→ Доверие (отзывы, до/после) 
→ Продажа (предложение, CTA)
```
