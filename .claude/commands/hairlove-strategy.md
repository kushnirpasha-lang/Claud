## АНАЛИЗ КОНКУРЕНТОВ — САЙТЫ МОНО-БРЕНДОВ

### Украинские бренды (прямые конкуренты)

**deeply.com.ua** — молодой бренд (2023), минималистичный дизайн, чистый UI.
Принцип: простые составы, доступные цены, профессиональное + домашнее применение.
*Что взять:* чистота дизайна, акцент на составе.

**jnowaprofessional.ua** — большой украинский профессиональный бренд (ACME Group).
Широкая линейка, B2B раздел, обучение. Более "тяжёлый" сайт.
*Что взять:* структура B2B раздела, раздел для мастеров.

**hairsekta.com** — российский бренд, полный цикл производства.
Профессиональная политика продаж (подтверждение профессионального статуса для скидок).
*Что взять:* SEKTA PRO программа — идея для HairLove Pro (условия для салонов).

### Международные ориентиры (к которым тянуться)

**K18hair.com** — победитель в дизайне среди hair brands. Чёрный/белый, product-first, минимум слов максимум визуала. Один продукт = одна страница, чётко и без воды.

**Olaplex.com** — системный подход: номера продуктов, понятная навигация, сильный раздел "как использовать". Education-heavy.

**Kerastase.com** — люкс, но сложный. Для HairLove пока избыточно.

### Вывод для сайта HairLove
Позиционирование: между deeply (простота) и K18 (премиум минимализм).
Итальянское производство = дизайн должен это чувствоваться.

## САЙТ — СТРУКТУРА И ТРЕБОВАНИЯ

### Цель сайта (этап 1)
Не интернет-магазин. **B2B витрина + бренд-презентация.**
Дистрибутор/салон приходит → видит серьёзный бренд → оставляет заявку.

### Структура страниц

```
Главная
├── Hero: бренд + ключевое УТП (Made in Italy, 4 продукта)
├── Продукты (краткий обзор всех 4)
├── "Почему HairLove" (Made in Italy, составы, результат)
└── CTA: стать партнёром / заказать тестеры

Продукты (отдельная страница каждого)
├── Большое фото флакона
├── Для кого / зачем
├── Состав (ключевые ингредиенты + их действие)
├── Как применять (пошагово)
└── Доступные объёмы + цены

B2B / Партнёрам
├── Условия сотрудничества
├── Прайс (дистрибутор / салон / мастер)
├── Тестеры — как получить
└── Форма заявки (имя / город / тип бизнеса / контакт)

О бренде
├── История (Made in Italy, кто стоит за брендом)
├── Философия (без лишнего, рабочие составы)
└── Instagram лента (виджет)

Контакты
└── Форма / мессенджеры / Instagram
```

### Дизайн-требования
- Цветовая схема: из этикеток (бирюзово-белый + зелёный + нежно-розовый)
- Шрифт: sans-serif, современный, читаемый
- Фото: флаконы на белом (уже есть 3 шт.) + lifestyle (нужно создавать)
- Языки: украинский основной, русский опционально
- Mobile-first (большинство B2B тоже смотрят с телефона)

### Технология
- **Shopify** или **Tilda** — для быстрого старта без разработчика
- Shopify: если планируется онлайн-продажа (B2C)
- Tilda: если пока только витрина + форма заявки (проще, дешевле)
- **Рекомендация:** Tilda на старте → Shopify при добавлении онлайн-оплаты

| Рубрика | Частота | Формат | Пример темы |

### Ближайшие посты (очередь)
1. Представить термозащиту (вышла вторая партия — момент для поста)
2. "Made in Italy — что это значит для твоих волос"

4. Ламелярная маска — объяснить двойное применение (бальзам vs маска)

### Голос бренда
Живой, тёплый, экспертный — как совет от подруги-профессионала.
Без штампов. Конкретика: ингредиенты, ощущения, результат.

## КОНТЕНТ — ФОТО/ВИДЕО (что есть и что нужно)

### Есть сейчас
- 3 фото флаконов на белом фоне (крем-спрей, термозащита, ламелярная маска)

### Что нужно создать (ТЗ для фотографа/дизайнера)

**Тип 1: Предметная съёмка**
- Флаконы на разных фонах (мрамор, дерево, ткань, цветы)
- Флаконы в руках (женская рука с ухоженными ногтями)
- Флаконы в ванной / на туалетном столике (lifestyle)

**Тип 2: Результат на волосах**
- До/после применения каждого продукта
- Блестящие ухоженные волосы (результат маски)
- Волосы у утюжка (термозащита в действии)

**Тип 3: Ингредиенты**
- Масло арганы, шёлк, авокадо — красивые визуалы ингредиентов

**Тип 4: B2B контент**
- Флаконы на полке салона / стойке мастера
- Мастер работает с продуктом

*Пока фото нет — используем белый фон + хороший текст + графика/шаблоны Canva*
