## Контекст и исходная точка

HairLove — действующий бренд с подтверждёнными продажами (~3500–4000 шт. флагмана), но без системной маркетинговой инфраструктуры. Нет сайта, нет брендбука, нет структурированного B2B-онбординга, реклама не запущена. Задача — выстроить эту инфраструктуру поэтапно, не тормозя текущие продажи.

## Критический путь

```
[Фаза 0: Основа]
  planning → дорожная карта
  competitors → данные рынка        ─┐
  strategy → позиционирование        ├─► [Фаза 1: Идентичность]
                                     │
[Фаза 1: Идентичность]               │
  brandbook → визуал + голос         ┤
  copy → базовые тексты              ├─► [Фаза 2: Каналы]
  text-editor → вычитка              │
                                     │
[Фаза 2: Каналы]                     │
  site → B2B витрина                 ┤
  instagram → контент-план           ├─► [Фаза 3: Трафик]
  growth → онбординг дистрибуторов   │
                                     │
[Фаза 3: Трафик]
  ads → Meta/Google кампании
```

**Блокеры по цепочке:**
- `strategy` блокирует `brandbook` и `copy`
- `brandbook` блокирует `site`, `instagram`, `ads`
- `copy` (+ text-editor) блокирует финальный контент сайта и постов
- `site` блокирует запуск платной рекламы
