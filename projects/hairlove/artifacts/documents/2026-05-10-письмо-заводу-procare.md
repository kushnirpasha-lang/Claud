## Перечень документов, которые нам необходимы

Для прохождения процедуры МОЗ нам нужен полный пакет документации по каждой из **трёх формул** (если документы применимы по формуле, а не по SKU — достаточно одного комплекта на формулу; уточните, пожалуйста, этот момент).

**1. Certificate of Free Sale (CFS) / Экспортный сертификат**  
Официальный документ, подтверждающий, что продукт законно продаётся в стране производства (Италия). Выдаётся уполномоченным органом (обычно — торговая палата или Министерство здравоохранения Италии). Необходим для каждой формулы.

**2. Сертификат GMP производителя**  
Действующий GMP-сертификат предприятия-производителя (ISO 22716 или эквивалент). Подтверждает, что производство соответствует международным стандартам качества. Желательно с переводом или хотя бы с нотариально заверенной копией.

**3. Паспорт безопасности (SDS / MSDS) для каждой формулы**  
Safety Data Sheet по стандарту REACH / EU 1272/2008. По одному документу на каждую из трёх формул. Нам нужна актуальная версия (не старше 3 лет).

**4. Техническая спецификация / Product Specification**  
Полное техническое описание продукта: физико-химические показатели, органолептические характеристики, условия хранения, срок годности, pH, вязкость и другие параметры контроля качества. По одному документу на формулу.

**5. Список INCI ингредиентов (официальный, подписанный)**  
Полный перечень ингредиентов в порядке убывания концентрации по номенклатуре INCI. Документ должен быть подписан уполномоченным представителем завода. Это ключевой документ для МОЗ — просим отнестись к его оформлению особенно внимательно.

**6. Результаты Challenge Test (Preservation Efficacy Test)**  
Тест на эффективность консервирующей системы в соответствии с Регламентом ЕС 1223/2009 и EN ISO 11930. Это **обязательное требование МОЗ Украины**, которое нередко является узким местом при регистрации. Если тест не проводился — просим сообщить немедленно, чтобы мы могли организовать его в украинской или европейской аккредитованной лаборатории.

**7. Результаты Stability Test (Тест стабильности)**  
Протокол испытаний на стабильность продукта (ускоренное и/или реальное время). Должен подтверждать заявленный срок годности. По одному протоколу на формулу.

**8. Дерматологический тест / Тест переносимости кожей**  
Результаты клинического или дерматологического тестирования продукта (patch test, dermatologist-tested). Если проводилось независимой лабораторией — нужно указать её название и аккредитацию. По одному документу на формулу.

**9. Документы, подтверждающие заявленные свойства (Claims Substantiation)**  
Обоснование маркетинговых заявлений продукта (например, «защита до 230°C», «20 действий в 1», «ламеллярный эффект» и т.д.) — клинические данные, инструментальные исследования или ссылки на стандарты. Это требование МОЗ стало обязательным; без него регистрация невозможна.

**10. Certificate of Analysis (CoA) для каждой партии**  
Сертификат анализа готовой продукции с результатами контроля качества. Если CoA выдаётся для каждой партии — предоставьте актуальный образец по каждой формуле. Нам важно понимать, какие параметры туда включены.

**11. Макеты этикеток на украинском языке**  
Финальные или предварительные макеты этикеток для всех 9 SKU с переводом всех обязательных элементов на украинский язык. Если перевод ещё не готов с вашей стороны — сообщите, мы подготовим его самостоятельно и направим вам на согласование.

**12. Документация по безопасности упаковочных материалов**  
Подтверждение соответствия первичной упаковки (флаконы, насосы, крышки) требованиям безопасности: сертификаты на контакт с косметической продукцией, декларации производителей упаковки. По одному комплекту на тип упаковки.

**13. Производственная документация (описание технологического процесса)**  
Краткое описание производственного процесса (Manufacturing Process Description): основные этапы производства, условия смешивания, контрольные точки, оборудование. Этот документ используется МОЗ при технической экспертизе досье.
