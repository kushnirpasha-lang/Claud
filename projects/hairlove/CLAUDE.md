# HairLove — карта проекта

Итальянский бренд профессиональной косметики для волос. Made in Italy.
Владелец: Павел (@Pavel_Kus). Язык общения: русский.

## База знаний (читай перед любой задачей)
knowledge/компания.md, knowledge/продукт.md, knowledge/бренд.md,
knowledge/аудитория.md, knowledge/метрики.md, knowledge/глоссарий.md

## Агенты
coordinator → планирует и маршрутизирует
planning → задачи и приоритеты
competitors → анализ рынка и конкурентов
strategy → стратегические решения
instagram → контент-план и публикации
copy → тексты для всех каналов
text-editor → вычитка и редактура
ads → Meta/Google реклама
site → сайт и B2B витрина
growth → рост аудитории и партнёрства

## Правила
1. Перед работой — читай knowledge/
2. Каждый шаг → событие: bash ~/.claude/событие.sh hairlove <агент> <тип> "<сообщение>"
3. Каждый шаг → строка в progress.log
4. Артефакты → artifacts/<домен>/ГГГГ-ММ-ДД-<тема>.md
5. knowledge/ — read-only для агентов, меняет только Павел
