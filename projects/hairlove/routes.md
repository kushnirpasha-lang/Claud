# Маршруты между агентами

Координатор читает этот файл и выбирает маршрут по описанию задачи.

---

## МАРШРУТ: запуск компании — Фаза 0 (Основа)

**Когда использовать:** старт проекта, первичная аналитика, нет базовых данных.
**Параллельный запуск:**
1. `planning` → дорожная карта, приоритеты, сроки запуска
2. `competitors` → анализ рынка: позиционирование, каналы, цены конкурентов
3. `strategy` → позиционирование HairLove, B2B/B2C фокус, ключевые решения

**Отчёт → координатор → утверждение → переход к Фазе 1**

---

## МАРШРУТ: запуск компании — Фаза 1 (Идентичность)

**Когда использовать:** после утверждения Фазы 0, есть стратегия и данные конкурентов.
**Параллельный запуск:**
1. `brandbook` → визуал, цвета, типографика, голос бренда (на основе strategy)
2. `copy` → базовые тексты для всех каналов (на основе strategy + brandbook)

**Последовательно после:**
3. `text-editor` → вычитка и контроль качества всех текстов

**Отчёт → координатор → утверждение → переход к Фазе 2**

---

## МАРШРУТ: запуск компании — Фаза 2 (Каналы)

**Когда использовать:** после утверждения Фазы 1, есть брендбук и базовые тексты.
**Параллельный запуск:**
1. `site` → B2B витрина на Tilda (структура + контент)
2. `instagram` → контент-план, первые публикации, визуал
3. `growth` → онбординг дистрибуторов, старт Одесса (салоны), тестеры

**Отчёт → координатор → утверждение → переход к Фазе 3**

---

## МАРШРУТ: запуск компании — Фаза 3 (Трафик)

**Когда использовать:** сайт и Instagram готовы, каналы активны.
**Последовательно:**
1. `ads` → запуск Meta/Google рекламы (таргетинг на B2B и B2C)

**Отчёт → координатор → итоги запуска**

---

## Схема зависимостей агентов

```
                    ┌─────────────────────┐
                    │     COORDINATOR     │
                    │  (точка входа/выхода)│
                    └──────────┬──────────┘
                               │
           ┌───────────────────┼───────────────────┐
           ▼                   ▼                   ▼
       [planning]         [competitors]        [strategy]
           │                   │                   │
           └───────────────────┴───────────────────┘
                               │
           ┌───────────────────┼───────────────────┐
           ▼                                       ▼
       [brandbook]                             [copy]
                               │
                          [text-editor]
                               │
                          [humanizer]  ← публичные тексты
                               │
           ┌───────────────────┼───────────────────┐
           ▼                   ▼                   ▼
        [site]            [instagram]           [growth]
                               │
                             [ads]
```

## Таблица зависимостей

| Агент | Получает от | Отдаёт в |
|---|---|---|
| **coordinator** | все агенты | все агенты |
| **planning** | coordinator | coordinator, strategy |
| **competitors** | coordinator | strategy, ads, growth |
| **strategy** | planning, competitors | site, instagram, ads, growth, brandbook |
| **brandbook** | strategy | site, instagram, copy, ads |
| **copy** | strategy, brandbook | text-editor → site, instagram, ads |
| **text-editor** | copy | humanizer (публичные тексты) |
| **humanizer** | text-editor | site, instagram, ads |
| **site** | strategy, brandbook, copy | ads, growth |
| **instagram** | strategy, brandbook, copy | growth |
| **ads** | site, competitors, strategy | growth |
| **growth** | site, instagram, ads | coordinator |

## МАРШРУТ: запуск рекламной кампании

**Когда использовать:** запустить рекламу, новый рекламный пакет, тест креативов.
**Цепочка:**
1. `competitors` → анализ позиционирования и креативов конкурентов
2. `strategy` → выбор сегмента и направления
3. `ads` → разработка кампании и текстов
4. `text-editor` → вычитка креативов
5. `humanizer` → антиИИ-проход перед публикацией
6. ✅ готово к публикации

---

## МАРШРУТ: контент в Instagram

**Когда использовать:** контент-план, серия постов, рилс, сторис.
**Цепочка:**
1. `strategy` → задаёт тему и угол
2. `instagram` → идея и контент-план
3. `copy` → финальные тексты
4. `text-editor` → вычитка
5. `humanizer` → антиИИ-проход перед публикацией
6. ✅ готово

*(опционально на шаге 1 — competitors для отстройки)*

---

## МАРШРУТ: обновление сайта

**Когда использовать:** новый лендинг, обновление текстов, новая страница.
**Цепочка:**
1. `strategy` → направление и ключевое послание
2. `site` → структура и черновики
3. `copy` → финальные тексты
4. `text-editor` → вычитка
5. `humanizer` → антиИИ-проход перед публикацией
6. ✅ готово

---

## МАРШРУТ: полный месячный цикл

**Когда использовать:** `/full-cycle` — комплексная работа на месяц.
**Цепочка:**
1. `competitors` → анализ активности за прошлый месяц
2. `strategy` → направление месяца
3. Параллельно:
   - `ads` → кампании → `text-editor` → `humanizer`
   - `instagram` → `copy` → `text-editor` → `humanizer`
   - `site` → `copy` → `text-editor` → `humanizer`
4. `coordinator` → сводка в `artifacts/_summary/`
5. ✅ готово

---

## МАРШРУТ: исследование рынка

**Когда использовать:** понять рынок, найти нишу, принять решение на основе данных.
**Цепочка:**
1. `planning` → формулирует, что исследовать
2. `competitors` → собирает данные
3. `strategy` → решение, выбирает следующего:
   - продвижение → ads
   - органика → growth или instagram
   - точка входа → site
   - коммуникация → copy
4. `text-editor` (если итогом был текст)
5. `humanizer` (если текст идёт в публикацию)
