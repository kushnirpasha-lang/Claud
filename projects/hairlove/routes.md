## ЗАКОН ПРОЕКТА — самопроверка

Перед запуском любого агента координатор разделяет в брифе ✅ проверено по источнику vs ⚠️ предположение. Тяжёлые задачи (деньги/закон/решение Павла/необратимое) — глубокий протокол. ## МАРШРУТ: Сертификация / юридический

**Когда использовать:** сертификация МОЗ/MOH, договоры, оферта, КП, нотификация, проверка пакета завода, любой официальный/регуляторный документ.
**Цепочка:**
1. `documents` → юр-анализ, дорожная карта, проверка по первоисточнику (или явная пометка реконструкции при 403)
2. `coordinator` → если нужны факты-решения Павла — поднять до Павла ПЕРЕД действием (Уровень 2)
3. ✅ артефакт в `artifacts/documents/`

**Право отклонить бриф:** если бриф противоречит первоисточникам (напр. «завод Италия») — `documents` останавливается и поднимает противоречие, не выдаёт гладкий ответ.

## МАРШРУТ: контент в Instagram

**Когда использовать:** контент-план, серия постов, рилс, сторис.
**Цепочка:**
1. `strategy` → тема и угол *(опц. `competitors` для отстройки)*
2. `instagram` → идея и контент-план
3. `visual` → промпты на фото под посты (если нужны новые изображения)
4. `copy` → финальные тексты
5. `text-editor` → вычитка
6. `humanizer` → антиИИ-проход перед публикацией
7. ✅ в очередь автопостинга (`artifacts/instagram/posting-queue.json`, ветка `claude/hairlove`)

---

## МАРШРУТ: визуал / генерация изображений

**Когда использовать:** нужны промпты под фото (Instagram, сайт, КП), переделка после неудачной генерации, визуальный концепт кампании, серия single-style.
**Цепочка:**
1. `brandbook` → визуальные правила (если концепт новый)
2. `visual` → готовые промпты под image-AI
3. ✅ промпты → внешняя генерация → канал назначения

---

## МАРШРУТ: обновление сайта

**Когда использовать:** новый лендинг, обновление текстов, новая страница.
**Цепочка:**
1. `strategy` → направление и ключевое послание
2. `site` → структура и черновики
3. `visual` → промпты на изображения (если нужны)
4. `copy` → финальные тексты
5. `text-editor` → вычитка
6. `humanizer` → антиИИ-проход
7. ✅ готово

---

## МАРШРУТ: запуск рекламной кампании

**Когда использовать:** реклама, новый рекламный пакет, тест креативов. *(ads неактивен — только по явной команде Павла)*
**Цепочка:**
1. `competitors` → анализ креативов конкурентов
2. `strategy` → сегмент и направление
3. `ads` → кампания и тексты
4. `text-editor` → вычитка
5. `humanizer` → антиИИ-проход
6. ✅ готово к публикации

---

## МАРШРУТ: исследование рынка / B2B-решение

**Когда использовать:** понять рынок, найти нишу, B2B-старт (Одесса), решение на данных.
**Цепочка:**
1. `planning` → что именно исследовать
2. `competitors` → данные рынка
3. `strategy` → решение, выбор следующего:
   - продвижение → `ads`
   - органика → `growth` / `instagram`
   - точка входа → `site`
   - коммуникация/скрипты → `copy`
   - юр./договор с дистрибутором → `documents`
4. `text-editor` (если итог — текст)
5. `humanizer` (если текст в публикацию)

---

## МАРШРУТ: полный месячный цикл

**Когда использовать:** `/full-cycle` — комплексная работа на месяц.
**Цепочка:**
1. `competitors` → анализ активности за прошлый месяц
2. `strategy` → направление месяца
3. Параллельно:
   - `instagram` → `visual` → `copy` → `text-editor` → `humanizer`
   - `site` → `copy` → `text-editor` → `humanizer`
   - `documents` → статус сертификации, что закрыть в этом месяце
   - `ads` → кампании → `text-editor` → `humanizer` *(если активирован)*
4. `coordinator` → сводка в `artifacts/_summary/`
5. ✅ готово
