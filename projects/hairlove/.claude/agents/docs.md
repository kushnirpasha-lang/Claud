## ТРЁХЭТАПНЫЙ ПРОЦЕСС СЕРТИФИКАЦИИ

### Этап 1 — Лабораторные испытания (~$1,600 USD)
Испытания в аккредитованной лаборатории:
- Микробиологические показатели
- Физико-химические показатели
- Испытания на безопасность

### Этап 2 — Получение документов (~$700 + 20,000–80,000 UAH)
- CPSR (Cosmetic Product Safety Report) — оценка безопасности токсикологом
- PIF (Product Information File) — досье продукта
- GMP-сертификат (или декларация завода о соответствии)
- Лабораторные протоколы
- Тесты стабильности

### Этап 3 — Нотификация в е-Косметика (~$200 / 10 групп)
- Уведомление через портал е-Косметика МОЗУ
- Назначение Ответственного лица (RP)
- Публикация Product Information File

## СТАТУС ДОКУМЕНТОВ ПО СЛОЖНОСТИ

### Легко получить ✅
- Технический регламент (публичный)
- Описание продукта / этикетка
- Декларация о соответствии (производитель выдаёт)
- GMP — возможна декларация от завода вместо ISO-сертификата
- Список ингредиентов (INCI) — есть у производителя
- Инструкция по применению

### Средней сложности ⚠️
- Лабораторные испытания — нужно найти аккредитованную лабораторию, занимает 4–8 недель
- Нотификация е-Косметика — технически несложно, но требует всех остальных документов
- Назначение RP — нужно юридически оформить ответственное лицо

### Проблемные документы 🚨
- **CPSR (оценка безопасности)** — токсикологу нужен полный состав формулы, но завод держит формулу в секрете → возможный блокер всей сертификации
- **Тесты стабильности** — стандарт: 4–5 месяцев (12 месяцев при 25°C). До дедлайна 03.08.2026 ОСТАЛОСЬ ~3 МЕСЯЦА → не успеть стандартным путём. Решение: **ускоренное тестирование (28 дней при 40°C)** — признаётся лабораториями как эквивалент

## КРИТИЧЕСКАЯ ПРОБЛЕМА: ДЕДЛАЙН

**Сегодня: май 2026. Дедлайн: август 2026. Осталось: ~3 месяца.**

Стандартное тестирование стабильности (4–5 мес.) не укладывается в сроки.

**Срочные действия:**
1. Немедленно запустить ускоренное тестирование стабильности (28 дней при 40°C)
2. Запросить у завода полный состав для CPSR (или оформить NDA с токсикологом)
3. Найти токсиколога для CPSR параллельно с испытаниями

## АЛЬТЕРНАТИВА: WHITE LABEL / ГОТОВАЯ ФОРМУЛА

**Идея:** купить уже сертифицированную формулу в лаборатории (напр. biolab.com.ua), использовать их документы для нотификации.

**Плюсы:**
- Документы уже готовы (экономия 3–4 мес.)
- Стабильность уже доказана
- CPSR уже написан
- Дедлайн август 2026 — реально успеть

**Минусы:**
- Нужно чтобы формула точно совпадала с продуктом на рынке (или переходить на их формулу)
- Потеря уникальности итальянской формулы как УТП
- Зависимость от другого поставщика
- Возможна потеря доверия если Максим/завод узнают о смене формулы

**Рекомендация:** уточнить в biolab.com.ua — есть ли у них готовые сертифицированные формулы для крем-спрея и термозащиты. Если да — оценить совместимость с текущим продуктом.
