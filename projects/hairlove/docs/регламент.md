# Регламент работы команды

## Артефакт (отчёт агента)

**Имя файла:** `artifacts/<домен>/ГГГГ-ММ-ДД-<тема>.md`

**Структура:**
```yaml
---
agent: competitors
date: 2026-05-06
task: Анализ Q2 активностей конкурентов
inputs:
  - knowledge/аудитория.md
  - knowledge/продукт.md
  - artifacts/competitors/2026-04-01-base-scan.md
status: ready
hand_off_to: [strategy, ads]
project: <имя_проекта>
---
```

Затем:
```markdown
# Краткое резюме (для других агентов)
2-3 абзаца. Главное и следствия.

# Полный анализ
...

# Рекомендации для смежных агентов
- strategy: учесть, что конкурент X сместился в нишу Y
- ads: усилить таргет на сегмент Z

# Источники
- ссылка 1
```

## Журнал передач — handoffs/pending.md

Append-only. Каждая передача — одна строка:
```
ГГГГ-ММ-ДД ЧЧ:ММ | от → к | путь_к_артефакту | "короткий комментарий"
```

Пример:
```
2026-05-06 14:00 | coordinator → competitors | handoffs/2026-05-06_14-00.md | "анализ конкурентов в нише X"
2026-05-06 14:30 | competitors → strategy | artifacts/competitors/2026-05-06-конкуренты.md | "учесть сдвиг конкурентов"
```

```bash

```

Типы: `start | step | complete | handoff | error | wait`
