## Суть проекта

MikhtavAI — Telegram-бот + веб-дашборд для русскоговорящих (и англоговорящих) репатриантов в Израиле.  
Помогает понимать письма и уведомления от госорганов на иврите.

**Проблема:** репатриант получает 20-50 писем/год от госорганов на иврите.  
Без понимания → штрафы, потеря льгот, испорченная кредитная история.  
Консультант стоит 200-400₪ за одно письмо. Google Translate не даёт контекст и дедлайны.

**Решение — три сценария:**
- **A. Фото письма** (Битуах Леуми, налоговая, муниципалитет, банки, суды, армия)
- **B. Скриншот приложения** (Maccabi, Clalit, банки, gmail на иврите)
- **C. Помощь со звонками** (скрипт до / расшифровка после) — Итерация 2, не MVP

**Что получает пользователь за 30-60 сек:**
1. От кого письмо
2. Срок ответа
3. Уровень важности
4. Объяснение сути на простом русском/английском
5. Конкретные шаги что делать
6. Шаблон ответа на иврите (для типовых случаев)
7. Кнопки: задать вопрос / сохранить в архив

**Юридический дисклеймер:** продукт даёт образовательную информацию, не юридическую консультацию.

## Технический стек

**MVP (сейчас):**
- Runtime: Node.js 22
- Language: TypeScript strict mode
- Telegram: grammy framework
- AI: Anthropic Claude API (`claude-sonnet-4-6`)
- Логгер: pino
- Валидатор: zod
- Package manager: pnpm 11

**Через 2-3 недели:**
- БД: Supabase (PostgreSQL + Auth + Storage)
- Платежи: Stripe + Tranzila
- Email: Resend
- Мониторинг: Sentry + Better Stack
- Веб: Next.js 14 + Tailwind + shadcn/ui
