# Глобальные правила (для любого проекта)

## Язык
- Всё мне в терминал — на русском.
- Английский — только команды, пути, имена файлов и переменных.

## Прозрачность
- Перед нетривиальным действием — объясни на русском, что делаешь.
- После действия — что получилось.
- Молчать во время длинных операций нельзя — пиши промежуточный статус в progress.log и шли событие в дашборд.

## Дисциплина
- Уточняй до 95% уверенности в требованиях — НО если пользователь сказал «делай всё и меня не спрашивай», выполняй без уточнений.
- Архитектурные решения — только с явного согласия пользователя.
- Перед изменениями в >1 файле — Plan Mode (если пользователь не сказал обратное).
- На 60-70% контекста — предложи /compact. На смене темы — /clear.

## Изоляция
- Никогда не выходи за корень текущего проекта.
- Никогда не упоминай и не запрашивай содержимое других проектов.
- Кросс-проектные задачи — только через пользователя.

## События дашборда
- На любое значимое действие — отправляй событие через `~/.claude/событие.sh`.
- Если скрипт недоступен — не падай, просто пиши в progress.log.
- Дашборд: http://188.166.67.237/agents

## Git
- Авто-коммиты на handoff/report/завершение маршрута настроены через hooks.
- Push в GitHub — только с явного подтверждения пользователя.

## Режим «делай всё»
Если пользователь написал «делай всё», «делай и не спрашивай» или аналог — выполняй весь объём задачи без промежуточных подтверждений. Сообщай только о ключевых результатах.
